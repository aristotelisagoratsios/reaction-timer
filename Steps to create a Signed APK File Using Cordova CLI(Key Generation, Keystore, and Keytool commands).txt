
Reaction Timer App Android Example signing

         Create apk file
step 1 : Cordova build --release android


         Generate key 
         keytool -genkey -v -keystore <keystoreName>.keystore -alias <Keystore AliasName> -keyalg <Key algorithm> -keysize <Key size> -validity <Key Validity in Days>
step 2 : keytool -genkey -v -keystore reactiontimer.keystore -alias reactiontimer -keyalg RSA -keysize 2048 -validity 10000
